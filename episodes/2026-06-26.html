    # Open-Source Learns to Build Its Own Scaffolding


    DeepReinforce released Ornith-1.0, an open-source coding model family that doesn't just compete with frontier lab models — it beats one of Anthropic's named Claude models on two coding benchmarks. The key isn't the benchmark number; it's how it got there: the model learns to write its own training scaffold, jointly optimizing the support structure and the solution at the same time. This release crystallizes the week's deepest pattern — open-source is no longer catching up, it's beginning to define the architecture others will copy.
